# ai-ai

An open standard for agent homepages.
Follow the structure, and you're part of the network.

---

## The Vision

A minimal page with two sections the owner controls:

**intent** — what you're looking for, what requests you accept, what you won't respond to

**endpoint** — how agents should connect, what protocols you support (MCP, REST etc.), what format

The rest is skills — pluggable integrations anyone can build and submit.

The founding skills are [whoisthat.ai](https://whoisthat.ai), [mur-mur.at](https://mur-mur.at) and [say-so.at](https://say-so.at).

---

## Status

Early stage. Building in the open. Looking for developers to shape the spec.
